vmf-api

The vmf-api is designed to give programmers and easy way to interact with .vmf files. These files are associated with Vavle's Source SDK Hammer Editor, a tool used to develop worlds that can be compiled and played in games that use the Source Engine.

Sample Code:

The following code generates a default vmf object and writes it to a .vmf file:


	    vmf_vmf sampleVMF = vmf_vmf_default();
	    FILE *file = fopen("output.vmf", "w");
	    vmf_fprint(file, sampleVMF);

.vmf Files

.vmf files are formatted in a similar fashion as JSON. They consist of labels, with each individual label consisting of many parameters, which may be strings (text, integers, floats, etc.) or sub-labels. These sub-groups are designated by curly braces {}.

Hammer Editor

The Valve Hammer editor is software for creating maps compatible with the Source Engine. The .vmf files handled by this API are meant to be completely compatible with the Hammer Editor. Information on using the Hammer Editor itself can be found at the Valve Developer Community wiki.
